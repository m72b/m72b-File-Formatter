Token Reference
{TrackNumber}Number at start of filename (zero-padded)
{TrackName}Main name portion after the number
{Artist}Text after first " - " separator
{Album}Text after second " - " separator
{Year}4-digit year found in filename
{ext}Original file extension (no dot)
{n}Auto counter: 01, 02, 03…
{original}Full original filename (no extension)
{UPPER}Original name in UPPERCASE
{lower}Original name in lowercase

How editing works in v2:
Files are renamed directly on disk — no downloading. Pick a folder, edit names inline or apply a format template, then click Save per row or Save All.
